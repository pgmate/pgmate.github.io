Postgres is the greatest tool of the modern development era.  
PGMate has the mission to provide with a **multi-purpose client** that let you:

- 🏋️‍♂️ work with your database
- 🤖 talk with your data
- 📋 create SQL workbook that you can share
- 🧪 bench your queries
- ✈️ manage migrations
- 🤓 read articles and learn

[![Open in Gitpod](https://gitpod.io/button/open-in-gitpod.svg)](https://gitpod.io/#https://github.com/pgmate/demo)

:::tip
PGMate is an MIT Licensed project and you can run it as a Docker Container:

```bash
curl -sL https://bit.ly/3ZAx19E | bash -s -- run
```
:::

Take a look at our [demo project](https://github.com/pgmate/demo) that helps you running PGMate in the Cloud via GitHub Codespaces, GitPod.io, or locally using Docker.

📖 Last but not least, PGMate is fully Open-Source: [collaborate in the repo](https://github.com/pgmate/pgmate).
