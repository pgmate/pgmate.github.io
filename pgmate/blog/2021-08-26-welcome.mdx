Postgres is the greatest tool of the modern development era.  
PGMate has the mission to provide with a **multi-purpose client** that let you:

- 🏋️‍♂️ work with your database
- 🤖 talk with your data
- 📋 create SQL workbook that you can share
- 🧪 bench your queries
- ✈️ manage migrations
- 🤓 read articles and learn

:::tip

PGMate is an MIT Licensed project and you can run it as a Docker Container:

```bash
docker run -p 8080:8080 pgmate/full
```

:::
